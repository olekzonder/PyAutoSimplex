# Решение задач линейного программирования с помощью симплекс метода
## Требуемые библиотеки: 
```
PyQt5;
python-docx;
NumPy
```
## Запуск: 
```
git clone https://github.com/olekzonder/pyautosimplex/
python pyautosimplex
```
